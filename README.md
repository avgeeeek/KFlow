# KineticFlow

**KineticFlow** is a reaction kinetics simulator split into a Frontend and Backend for distributed deployment.

## Structure

- **[Frontend](./frontend)**: Next.js 14 application (UI, Simulation Engine). Deploy to **Vercel**.
- **[Backend](./backend)**: Express.js server (API). Deploy to **Render**.

## Quick Start

### Frontend
```bash
cd frontend 
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```
